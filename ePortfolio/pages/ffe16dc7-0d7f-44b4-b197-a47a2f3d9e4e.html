Dreamweaver Final Project

 

Directions:  Using Dreamweaver and nothing else, create a web site that includes the following:
 
Two column layout with left sidebar, header, and footer
A header with Logo
Three web pages
Horizontal navigation bar using a Spry menu bar
At least one in page anchor link (ex. return to top)
At least two external links
At least three images on each page, including one in the header
A table with data
Modify you site with CSS in an external style sheet
At least two separate lists (unordered and ordered)
At least one Spry widget
 
 

Web Development Final Project 



Directions:  Working with your team mate, use Notepad++ only to hand-code XHTML and CSS to design a web site that includes the following:

Two Column layout with logo in the header, left sidebar vertical navigation area, main-content area, and footer 
Three pages, modified with CSS
All pages linked to an External Style Sheet
Navigation links between all pages
At least 2 external links
At least 2 images per page, not including logo
At least 2 separate lists (unordered & ordered)
At least 1 form
1 Audio and 1 Video Link
 

 This was my first 3 page website, so please be kind with the critique!
